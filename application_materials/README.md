## Application Materials Layout

- `Archive/`: software-engineering archive templates and older reference documents.
- `generated/`: generated outputs and normalized job-description artifacts.
- `job_descriptions/`: source job-description files used for tailoring and comparison.
- `reference_examples/customer_facing_technical/`: current live customer-facing template pair used by the generator.
- `reference_examples/technical_support/`: technical-support resume and cover-letter examples for future template comparison.
- `reference_examples/technical_writing/`: technical-writing resume and cover-letter examples for future template/design comparison.

The generator currently uses the implementation-specialist pair in `reference_examples/customer_facing_technical/` as the shared base for adjacent customer-facing technical roles.
